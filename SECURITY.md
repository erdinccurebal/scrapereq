# Security Policy

This project is currently in early development (version 1.0.0) and is private.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.0   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Reporting a Vulnerability

As this is a private project in its early stages, please report any security concerns directly to the project maintainer.

Do not share any vulnerabilities outside the development team.  
External vulnerability reporting channels are not available at this time.

Thank you for helping keep the project secure.
